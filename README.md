# Fast Merge - GitLab 合并助手

一个专为 VS Code 设计的 GitLab 合并请求管理插件，提供直观的图形界面来简化代码合并流程，大幅提升你的合并效率。

## 🚀 功能特性

### 核心功能
- **支持分支间的快速合并请求**
- **支持多选功能的Cherry Pick 合并，大幅节省时间**
- **支持冲突检测**
- **支持快速打开mr链接**
- **支持一键复制所有mr链接**
- **支持历史记录**
- **选择commit列表可根据showHash配置显示/隐藏 commit的hash**
- **支持一键打开所有mr链接**
