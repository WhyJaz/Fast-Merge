## 安装包
进入根目录和webview-ui目录下，分别执行yarn命令，安装npm包

## 开发调试
在根目录下，执行npm run dev:webview
然后按F5启动扩展调试，src目录下的修改需要重载页面（windows按ctrl+r，mac按command+r）,webview-ui下修改则是实时热重载


## 插件打包
安装打包工具命令： npm i -g vsce
打包插件命令：vsce package
